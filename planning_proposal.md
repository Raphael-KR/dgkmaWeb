# 동국대학교 한의과대학 동문회 홈페이지 구축 기획서 (Ver 3.0)

## 1. 프로젝트 개요
*   **프로젝트명**: 동국대학교 한의과대학 동문회 (Dongguk University College of Korean Medicine Alumni Association) 웹사이트 구축
*   **목적**: 카카오톡 커뮤니티를 보완하는 **모바일 우선(Mobile-First)** 플랫폼 구축 + **비영리 민간단체(장학사업)** 지정을 위한 **PC 웹의 격식(Official Look)** 갖춤.
*   **핵심 철학**: DevOps 기반의 운영 자동화 (관리 포인트 최소화)

## 2. 회원 정책 및 등급
*   **회원 구분**:
    *   **회원 (Member)**: 홈페이지 가입을 완료한 동문 (회비 미납)
    *   **권리회원 (Rights Member)**: 동문회비를 납부 중인 회원 (매월 납부 확인 시 권리회원 유지, 체납 시 회원으로 강등)
*   **가입 프로세스 (자동화)**:
    1.  **카카오싱크(Kakao Sync)**를 통해 사용자 정보(이름, 전화번호 등) 수신
    2.  **동문 DB(내장)**와 대조 (이름 + 전화번호)
    3.  **일치 시**: 즉시 **자동 승인** 및 가입 완료
    4.  **불일치 시**: 가입 거부 및 "카카오톡 채널 문의" 버튼 안내

## 3. 데이터베이스 및 마이그레이션 전략
*   **DB 전략**: **DB 내장 및 일원화** (구글 시트 의존성 제거)
    *   **Alumni Table**: 전체 졸업생 데이터 (기준 데이터)
    *   **User Table**: 가입 회원 데이터
*   **레거시 데이터 마이그레이션**:
    *   **소스**: 네이버 밴드, 다음 카페
    *   **전략**:
        *   **1단계 (아카이빙)**: 주요 역사적 자료(사진, 중요 공지) 선별
        *   **2단계 (이관)**: '사진으로 보는 동문회 역사' 등 특화 게시판으로 수동/반자동 이관
        *   **3단계 (폐쇄/안내)**: 기존 커뮤니티에 홈페이지 통합 공지

## 4. 주요 기능 상세

### A. 동문회 헤리티지 (Heritage) - PC 뷰 중요
*   **목적**: 비영리 민간단체 지정 및 동문회의 정통성 강조
*   **구성**:
    *   **동문회장 인사말**: 현 회장의 비전과 인사말
    *   **동문회 연혁**: 타임라인 형태의 역사 소개
    *   **조직도**: 회장단, 이사회, 각 위원회 구조도
    *   **역대 동문회장**: 역대 회장 사진 및 재임 기간
    *   **사진으로 보는 역사**: 앨범형 게시판 (네이버 밴드/다음 카페 사진 자료 이관)

### B. 동문 주소록 (Directory)
*   **접근 권한**: **가입한 회원**만 열람 가능
*   **열람 제한**: 본인 소속 **동기회(기수)** 및 **지부(지역)**만 열람 가능 (가입 유도)

### C. 커뮤니티 (Community) - 통합 게시판
*   **구조**: 단일 DB + 태그 시스템 (`안내`, `자유`, `경조사`, `장터`, `임상정보`)
*   **운영**: Staff 권한자가 프론트엔드에서 직접 관리

### D. 경조사 알림 시스템 (Killer Feature)
*   **AI 기반 자동 생성**: 텍스트/링크 입력 → AI 파싱 → 부고장/초대장 생성
*   **One-Stop Service**: 알림 확인 → 문자 발송/화환 신청 → 결제
*   **카카오톡 알림톡**: 알리고 API 연동을 통한 실시간 부고 알림 발송
*   **문자 발송**: 알리고 API를 통한 장문/단문 SMS 발송
*   **화환/근조기 주문**: 알리고 문자를 통한 주문 접수 및 관리자 처리

### E. 회비 결제 시스템
*   **카카오페이 연동**: 온라인 회비 정기 결제 및 일회성 납부
*   **자동 알림**: 미납 회원 대상 자동 알림 발송 (알림톡/SMS)
*   **납부 내역**: 회원별 납부 내역 조회 및 영수증 자동 발급
*   **권리회원 관리**: 납부 상태에 따른 회원 등급 자동 업데이트

### F. 경조사 서비스 결제
*   **통합 결제**: 문자 발송, 화환 주문 등 경조사 서비스 통합 결제
*   **카카오페이 연동**: 간편 결제 지원
*   **결제 내역 관리**: 서비스별 결제 내역 조회 및 영수증 발급

## 5. 디자인 전략 (Responsive)
*   **Mobile**: 카카오톡 인앱 브라우저에서 앱처럼 느껴지는 UX (Bottom Navigation, 터치 친화적)
*   **PC**: 관공서/교육부 심사를 대비한 **격식 있고 신뢰감 주는 레이아웃** (Wide Header, Footer, 정돈된 Grid)

## 6. 기술 스택 (확정)
*   **Frontend**: **React (Vite + TypeScript)**
    *   Tailwind CSS로 모바일/PC 반응형 디자인 구현 용이
*   **Backend**: **Node.js (Express) + TypeScript**
    *   Frontend와 타입 공유(Zod/Drizzle)로 **AI 협업(Vibe Coding) 효율 극대화**
*   **Database**: **Replit (PostgreSQL)**
*   **Infrastructure**: **Replit**

## 7. 향후 확장 계획

### A. 화환/근조기 업체 API 직접 연동

*   향후 화환 업체 API 직접 연동 시 자동 주문 처리 가능

### B. 장학사업 관리 시스템
*   장학생 선발 프로세스 디지털화

*   장학생 관리 및 후속 지원 추적

### C. 고급 분석 및 리포팅
*   회비 납부율 통계 및 분석
*   경조사 서비스 이용 현황 분석
*   동문 참여도 분석 및 리포트
